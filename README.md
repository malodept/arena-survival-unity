# Arena Survival Unity

A simple arena survival game developed with Unity.

## Gameplay
- The player stays in the center of the arena.
- Obstacles spawn and move toward the player.
- Clicking on an obstacle shoots a bullet directly at it.
- Destroy obstacles to increase your score.
- If an obstacle reaches the player, HP is reduced.
- Game ends when HP reaches zero.

## Controls
- **Arrow Keys**: Move the player
- **Mouse Left Click**: Shoot at obstacles

## Features
- Obstacle spawning system
- Raycast-based shooting
- Score and HP system
- Game Over screen with final score
- Simple UI (score, HP, quit button)

## Built With
- Unity 6 (6000.3.1f1)
- C#
